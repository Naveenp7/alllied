Automated Attendance System

A smart attendance system leveraging OpenCV and AI-driven facial recognition (YOLO) for real-time ID card detection and proxy prevention. It uses LBPH for accurate logging, ensuring efficiency in educational and corporate environments.

AI Pose Estimation

An AI-powered pose estimation and calorie tracker for real-time gym workouts. Built using OpenCV and MediaPipe, it tracks exercises like bicep curls and provides feedback on form and calorie burn.

Intruder Detection

An AI-based security system that detects intruders using motion detection and facial recognition. Integrated with OpenCV and deep learning models, it sends real-time alerts to users.

Wedding Management

A comprehensive wedding planning tool powered by AI to manage guest lists, budgets, and schedules. Features include automated reminders and vendor coordination.

Budget Tracker

An AI-driven budget tracker that analyzes spending patterns and provides personalized financial advice. Built with machine learning to predict future expenses.

An AI-powered tool to monitor and analyze electricity usage in real-time. It provides insights to optimize consumption and reduce costs, using data visualization and predictive analytics.
